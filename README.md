### 🚀 A few things I've built

**TailorPay — Multi-tenant SaaS payroll platform**
Configurable payroll engine letting companies define custom pay rules and schedules without code, with schema-based tenant isolation. *(.NET, ASP.NET Minimal APIs, EF, PostgreSQL, React, Tailwind, Docker)*

**Chess Police — AI cheating detection** · *MSc thesis, grade 12*
Multi-model AI system that processed **280M+ games** using unsupervised learning and anomaly detection, reaching **100% precision** and **90.5% accuracy**. *(Python, TensorFlow, LSTM)*

**FSQL — Fluent SQL builder for C#/.NET** · *Open source*
Strongly-typed, chainable API for safe parameterized queries, published as a versioned NuGet package.

**Joker's Junction — Architecture study**
Built monolithic and microservices versions of the same system to compare scalability and design trade-offs. *(.NET, gRPC, RabbitMQ, Kubernetes)*
